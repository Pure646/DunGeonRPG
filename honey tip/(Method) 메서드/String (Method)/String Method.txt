.ToString()		: 객체를 문자열로 변환 (기본적이고 많이 사용되는 방법)

string.Format()		: 문자열 내에 변수를 포맷해서 삽입할 때 사용

String.Concat()		: 여러 문자열을 합칠 때 사용

String.Join()		: 배열이나 컬렉션을 구분자를 넣어서 합칠 때 사용

Convert.ToString()		: null 처리에도 안전하게 문자열로 변환

StringBuilder.Append()	: 문자열을 여러 번 변경할 때 효율적으로 사용

문자열 보간법 ($"{}"): 변수 값을 문자열에 직접 삽입할 때 가독성 좋고 간편
