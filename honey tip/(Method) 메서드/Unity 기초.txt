[ Awake() ]
- 가장 먼저 실행되는 함수
- 비활성화된 상태여도 실행됨.
- 주로 변수 초기화 혹은 싱글톤 패턴 설정 등에 사용된다.

[ OnEnable() ]
- 오브젝트가 활성화될 때마다 실행됨.
- 비활성화 후 다시 활성화하면 다시 실행됨
- SetActive(true) 로 다시 활성화될 때도 실행됨.
- 이벤트 리스너를 등록할 때 사용됨. (이벤트 구독)

[ Start() ]
- Awake() 실행 후, 첫 번째 프레임이 시작되기 전에 실행됨.
- 다른 오브젝트와 상호작용이 필요한 초기화에 사용됨.
- 스크립트가 처음 한 번만 실행됨 (오브젝트가 씬에서 처음 활성화될 때)
- 비활성화 후 다시 활성화해도 실행되지 않음.
- 씬이 실행될 떄 실행.

[ Update() ]
- 매 프레임마다 실행됨.
- 사용자 입력 처리, 애니메이션 업데이트 등에 사용

[ FixedUpdate() ]
- 물리 프레임마다 호출됨 (일반적으로 초당 50번, Time.fixedDeltaTime 기준)
- Rigidbody를 사용할 때는 Update()보다 FixedUpdate()가 더 적합
- 프레임 속도와 상관없이 일정한 간격으로 실행됨

[ LateUpdate() ]
- 모든 Update() 실행 후 호출됨
- Update()가 모두 실행된 후 마지막에 실행됨
- 카메라 추적, 애니메이션 후처리, UI 업데이트 등에 사용

[ OnDisable() ]
- 오브젝트가 비활성화될 때 실행됨
- 이벤트 리스너를 해제할 때 사용됨.

[ OnDestroy() ]
- 오브젝트가 삭제될 때 실행됨.
- 씬이 변경되거나 Destroy(gameObject); 가 호출될 때 실행됨.
- 정리 작업, 파일 저장 등에 사용됨.
