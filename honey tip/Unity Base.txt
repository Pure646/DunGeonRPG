[ Start() ]
-스크립트가 처음 한 번만 실행됨 (오브젝트가 씬에서 처음 활성화될 때)
-비활성화 후 다시 활성화해도 실행되지 않음

[ FixedUpdate() ]
- 물리 프레임마다 호출됨 (일반적으로 초당 50번, Time.fixedDeltaTime 기준)
- Rigidbody를 사용할 때는 Update()보다 FixedUpdate()가 더 적합
- 프레임 속도와 상관없이 일정한 간격으로 실행됨

[ LateUpdate() ]
- 모든 Update() 실행 후 호출됨
- Update()가 모두 실행된 후 마지막에 실행됨
- 카메라 추적, 애니메이션 후처리, UI 업데이트 등에 사용

[ OnEnable() ]
OnEnable()은 Unity에서 게임 오브젝트(GameObject)가 활성화될 때 자동으로 호출되는 생명 주기 함수야.
- 오브젝트가 활성화될 때 실행해야 하는 초기화 작업을 넣는 곳이라고 보면 돼!
- 오브젝트가 활성화될 때마다 실행됨
- 비활성화 후 다시 활성화하면 다시 실행됨
