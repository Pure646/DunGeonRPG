게임이 시작될 때 자동으로 호출되는 메서드를 지정
초기화 작업을 게임 시작 시에 자동으로 수행하고 싶을 때 사용된다.

ex)
[RuntimeInitializeOnLoadMethod]
static void RunOnStart()
{
    Debug.Log("게임이 시작되었습니다!");
}

[RuntimeInitializeOnLoadMethod]
1. 호출 시점			: 게임이 시작되기 전 (게임 오브젝트가 활성화되기 전에) 호출된다.
2. 메서드 유형			: 정적 메서드만 사용 가능
3. 주 용도 			: 게임의 초기화 작업, 설정, 전역적인 초기화
4. 인스턴스 객체 접근 가능 여부 	: 불가능 (정적 메서드이므로 인스턴스 객체에 접근할 수 없음)

Start()
1. 호출 시점			: 게임 오브젝트가 활성화된 후 첫번째 프레임에서 호출된다.
2. 메서드 유형 			: 인스턴스화된 메서드 사용 가능
3. 주 용도 			: 컴포넌트 초기화, 게임 오브젝트가 준비된 후 동작
4. 인스턴스 객체 접근 가능 여부 	: 가능 (컴포넌트나 오브젝트에 접근 가능)
