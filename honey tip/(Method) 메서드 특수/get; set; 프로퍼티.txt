public float number { get; set; }
[ 컴파일러 허용 ]

외부에서도 내부에서도 읽고 쓰기가 가능.
---------------------------------------------------------------------------------------------------------------------------------------------
public float number { get; private set; }
[ 컴파일러 허용 ]

외부에선 읽기만 가능하고, 내부에서만 쓰기 가능.
---------------------------------------------------------------------------------------------------------------------------------------------
public float number { private get; set; }
[ 컴파일러 허용 ]

읽기(get)는 private → 내부에서만 읽기 가능
쓰기(set)는 public → 외부에서 값 설정 가능
흔히 쓰이는 형태는 아니지만, 문법적으로 맞고 실제 컴파일 됩니다.
---------------------------------------------------------------------------------------------------------------------------------------------
public float number { private get; private set; }
[ 컴파일러 불가능 ]

프로퍼티는 public인데, get/set 모두 private이면 의미가 없어서 에러 발생.
---------------------------------------------------------------------------------------------------------------------------------------------
get 의 호출 개념
사용(읽기)이 되는 순간 바로 호출

ex)
public float Number {get; set;}

float x = Number;				// 이 때 get 호출
Debug.Log(Number);			// 이 때 get 호출

---------------------------------------------------------------------------------------------------------------------------------------------
set 의 호출 개념
저장되는 순간 바로 호출

ex)
public float Number {get; set;}

Number = 10;				// 이 때 set 호출
---------------------------------------------------------------------------------------------------------------------------------------------
get, set 에 의해서 그 값이 호출이 되면 먼저 프로퍼티가 발생시키고 그리고 호출하려거나, 저장하려는 내용이 나온다.
