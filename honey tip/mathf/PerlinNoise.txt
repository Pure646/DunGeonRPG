[ Mathf.PerlinNoise(float x, float y) ]

x	: X 좌표를 기준으로 퍼린 노이즈를 생성할 위치.
y	: Y 좌표를 기준으로 퍼린 노이즈를 생성할 위치.

- 퍼린 노이즈는 주변 값과의 관계를 고려하여 부드럽고 자연스러운 변화가 일어나도록 설계되어 있습니다
- 퍼린 노이즈는 일반적인 랜덤 수처럼 보일 수 있지만, 그 결과물은 비교적 예측할 수 있는 패턴을 가집니다. 
이는 자연적인 패턴을 만들 때 유용합니다.
- 퍼린 노이즈는 주기적인 성질을 가지고 있습니다. 즉, 값이 일정 범위 내에서 반복되는 형태를 보일 수 있습니다.

ex)
void Start()
{
    float x = 0.5f;  // x 좌표
    float y = 0.5f;  // y 좌표

    // 퍼린 노이즈 값을 계산
    float noiseValue = Mathf.PerlinNoise(x, y);

    // 노이즈 값 출력 (0에서 1 사이의 값)
    Debug.Log(noiseValue);
}
