?. 	[ null 조건 연산자 ]
객체가 null일 수도 있을 때 안전하게 접근하게 해주는 문법

ex)
GameObject obj = null;
Debug.Log(obj?.name);

위 코드에서 obj가 null이면 그냥 null 반환하고,
null이 아니면 name 프로퍼티 값을 반환해요.
---------------------------------------------------------------------------------------------------------------
? : 	[ 삼항 조건 연산자 ]
(조건) ? 참일 때 값 : 거짓일 때 값

ex)
int score = 80;
string result = (score >= 60) ? "합격" : "불합격";
---------------------------------------------------------------------------------------------------------------
?? 	[ null 병합 연산자 ]
왼쪽이 null이면 오른쪽 값을 사용

ex)
string name = inputName ?? "기본이름";
---------------------------------------------------------------------------------------------------------------
??=	[ null 병합 할당 연사자 ]
왼쪽이 null 일 때만 오른쪽 값 할당

ex)
inputName ??= "기본이름";
